| Key Combination | Action | Description |
| :--- | :--- | :--- |

## ü™ü Window Management

| Key Combination | Action | Description |
| :--- | :--- | :--- |
| **Alt + Tab** | **Cycle Windows** | Switch between windows |
| **Super + F** | **Fullscreen** | Toggle fullscreen mode |
| **Super + Shift + F** | **Float** | Toggle floating mode |
| **Super + P** | **Pseudo** | Toggle pseudotiling (dwindle) |
| **Super + Shift + I** | **Split** | Toggle split direction |
| **Super + Arrow Keys** | **Focus** | Move focus to another window |
| **Super + Ctrl + Arrow** | **Move** | Move window in that direction |
| **Super + Shift + Arrow** | **Resize** | Resize active window |
| **Super + Mouse Drag** | **Move/Resize** | Move (Left Click) or Resize (Right Click) |
